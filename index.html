NYC Sample Sale map aggregator

Link to map. Python script crawls www.thechoosybeggar.com for location data within the divs on the front page and stores a json string that is then used to create a map! Simple stuff but actually kinda useful when trying to acheive value in clothing. I run the script to update the json file periodically, I should really find a way to automate it to run on its own every few days...
